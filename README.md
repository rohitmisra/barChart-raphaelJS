barChart-raphaelJS
==================

A simple barchart library using RaphaelJS

  - Create simple barcharts
  - Multi color support
  - Landscape/portrait mode
  - Easy configuration
  - 




Installation
--------------

```sh
git clone https://github.com/rohitmisra/barChart-raphaelJS.git
cd barChart-raphaelJS
npm install


#for an illustration:

grunt connect:example

```
